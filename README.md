# Walky Talky Web

## Development setup
### Install dependencies
```
npm install
```

## Commands
### Run dev server
```
npm run start:dev
```

### Run Cypress
```
npm run start:cypress
```

### Run unit tests
```
npm run test:unit
```

### Run unit tests (watch)
```
npm run test:unit:watch
```

### Run build
```
npm run build
```
